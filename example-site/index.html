Design Tokenzier Integration Example

An example of integrating design tokens into your pipeline.

  1. To get started, install the Design Tokenizer plugin for Figma.
  2. Next, install Style Dictionary.
  3. This site is currently using the styles from the Style 1 page in the example library.
  4. Make changes to the styles (or jump over to the Style 2 page) and run Design Tokenizer to create new tokens.
  5. Save to style-dictionary/properties.
  6. Run "style-dictionary build" to transform the tokens into CSS variables.
  7. For this example, overwrite the :root body selector in example-site/css/styles.css with the contents of style-dictionary/build/tokens.css.
  8. Save files and refresh the page to see the changes!
