Design Tokenzier Integration Example

  • The starter project demonstrates one possible way to integrate design tokens into your pipeline.

  • To get started, install the Design Tokenizer plugin for Figma.
  • Next, install Style Dictionary.
  • This site is currently using Style 1 from the example library.
  • Update the design tokens by navigating to the Style 2 page and generate new design tokens using Design Tokenizer.
  • Save the tokens to style-dictionary/properties.
  • Run "style-dictionary build" to transform the tokens into CSS variables.
  • For this example, overwrite the :root body selector in example-site/css/skeleton.css with the contents of style-dictionary/build/tokens.css.
  • Save files and refresh the page to see the changes!
