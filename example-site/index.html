Design Tokenzier Integration Example

An example of integrating design tokens into your pipeline.

  1. To get started, install the Design Tokenizer plugin for Figma.
  2. Next, install Style Dictionary.
  3. Navigate to the style-dictionary/ folder of this project and run npm install to set up required dependencies.
  4. This project is using the current styles generated from the example design library. Open example-site/index.html to see what they look like.
  5. Make changes to the styles in the Figma file and run the Design Tokenizer plugin to create new tokens.
  6. Save the JSON file to style-dictionary/properties.
  7. In style-dictionary/ run "npm run build" to transform the tokens into CSS variables, saved to style-dictionary/build/tokens.css.
  8. For this example, overwrite the :root body selector in example-site/css/styles.css with the contents of style-dictionary/build/tokens.css.
